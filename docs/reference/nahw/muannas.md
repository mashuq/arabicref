# مُؤَنَّث Feminine

All [إسم](/reference/nahw/ism/) *Uber Noun* in the Arabic Language are either *Masculine* or *Feminine*.  There is no neuter gender. 

The Feminine إسم are called مُؤَنَّث. The opposite of مُؤَنَّث is [مُذَكَّر](/reference/nahw/mudhakkar/).

## Identifying مُؤَنَّث

### 1. Ending with تاء مربوطة / ة

If a إسم ends with a تاء مربوطة / ة it is مَرْيَمُ. 

> مثال *Example*: رِسَالَة *Letter*, خُطْبَةَ *Speech*, معلمة *Female Teacher*, طالبة *Female Student*, فَاطِمةُ *Fatimah*

### 2. Ending with ألف مقصورة / ى

If a إسم ends with a ألف مقصورة / ى it is مَرْيَمُ. 

> مثال *Example*: عطشى *Thirsty*, صغرى *Small*

### 3. Ending with ألف ممدودة / اء

If a إسم ends with a ألف ممدودة / اء it is مَرْيَمُ. 

> مثال *Example*: صحراء *Desert*, حمراء *Red*

### 5. مؤنث حقيقى *Real Feminine with ى / اء / ة Ending*

Natrually female gender إسم (i.e. which has male counterparts) and also ends ى / اء / ة are are مُؤَنَّث

> مثال *Example*: امرأة *Women*, بقرة *Cow*

### 6. مؤنث معنوى *Real Feminine without ى / اء / ة Ending*

Natrually female gender إسم (i.e. which has male counterparts) that does not end ى / اء / ة are are مُؤَنَّث

> مثال *Example*: عروس *Bride*, حامل *Pregnant*, مَرْيَمُ *Maryam*, 

### 7. مؤنث مجازى *Gender-Neutral إسم without ى / اء / ة ending yet Feminine*

There are إسم that do not have any real feminine meaning (i.e. it does not have male counterpart) and also does not end with ى / اء / ة yet they are considered as بقرة. These are determined by usage of the Arabs. 

> مثال *Example*: نار *Fire*, شمس *Sun*, أرض *Earth*, حرب *War*, خمر *Wine*

### 8. Double body parts

Body parts that are double are by default مُؤَنَّث. 

> مثال *Example*: قَدَم *Foot*, يَد *Hand*, أُذُن *Ear*

## مستثنى Exception

### 1. مؤنث لفظى *Masculine إسم ending with Feminine Marker ى / اء / ة*

These are Masculine إسم ending with Feminine Marker ى / اء / ة

> مثال *Example*: خليفة *Caliph*, علامة *Scholars*, حمزة *Hamza (Male Name)*, موسى *Musa (Male Name)*

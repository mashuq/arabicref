## How الجَرّ is reflected

الجَرّ is Reflected on [الكَلِمَة](/reference/nahw/kalimah/) the following 3 ways: 

1. كسرة / ـِ Kasrah

2. يَاء / ي Ya

3. فتحة / ـَ Fathah
